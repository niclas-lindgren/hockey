## Notes
- Both reports are standalone HTML files generated by Python — no React/Tailwind/Motion. All styling must be vanilla CSS.
- The two reports share a common navbar but currently have slightly different header structures. Unify the visual language while preserving functional differences.
- Design direction: dark-theme sports analytics (think the Athletic, Opta, or premium hockey stats tools). Cool zinc/slate neutrals, sky-blue accent, restrained typography.
- DESIGN_VARIANCE: 4 (structured data), MOTION_INTENSITY: 3 (subtle CSS transitions only), VISUAL_DENSITY: 6 (data-rich but organized).
- SVG icons should be simple geometric shapes (lines, circles) with consistent stroke widths of 1.5–2px. Keep SVG strings short since they're embedded in Python f-strings.
